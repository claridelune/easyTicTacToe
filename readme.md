# Easy Tic Tac Toe

**Clone Repo**

```
git clone --recursive https://github.com/claridelune/easyTicTacToe.git
```

**Compile**

```
./compile.sh
```

**Execute**

*Server Main*
```
./build/server_main/serverMain
```

*Server Trainer*
```
./build/server_trainer/serverTrainer
```
